# AGENTS.md

Instructions for AI coding assistants working on this project.

## Language

- **Default language**: English — always respond in English unless asked otherwise.
- All code, comments, commit messages, and documentation should be in English.

## Code Explanations

- **Keep it simple** — when the user asks for a code explanation, give a concise, plain-English summary.
- Focus on *what* the code does and *why*, not line-by-line breakdowns unless explicitly requested.
- Use short bullet points when helpful.

## Project Context

- **Stack**: Next.js 16 (App Router), Supabase, Tailwind CSS v4, MUI, Redux Toolkit, React Query
- **Purpose**: AI-powered job application engine — scrape job listings, score them against a resume, and generate cover letters.
